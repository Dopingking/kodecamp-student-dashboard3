# KodeCamp 6.0 — Stage 3: Student Enrollment Portal

This React application fulfills all requirements of the stage 3 task.

## Features
- Fetches student roster from Random User API.
- Controlled and uncontrolled form inputs.
- Live preview, validation, and enrollment.
- Component-based architecture with functional and class components.

## Setup & Run
```bash
npm install
npm run dev
```
